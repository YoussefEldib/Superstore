# 📊 Superstore Sales Analysis with SQL

## 🚀 Project Overview

This project explores the **Superstore Sales Dataset** using SQL to perform data exploration and generate business insights. The analysis focuses on sales performance, profitability, customer segments, product categories, and regional trends.

## 🎯 Objectives

* Explore the structure of the dataset.
* Analyze sales and profit performance.
* Identify loss-making transactions.
* Examine customer segments and product categories.
* Practice SQL querying, filtering, and sorting techniques.

## 🗂️ Dataset Information

The Superstore dataset contains transactional sales records, including:

* Order ID
* Customer Name
* Product Category
* Sales
* Profit
* Customer Segment
* Region

## 🔍 Analysis Performed

### Data Exploration

* Displayed all records from the dataset.
* Retrieved specific columns for focused analysis.
* Identified unique product categories.

### Data Filtering

* Orders with negative profit.
* Orders from the West region.
* Corporate customer transactions.
* Technology category products.

### Data Sorting

* Highest sales transactions.
* Lowest profit transactions.
* Top-performing sales records.

## 🛠️ SQL Concepts Used

* `SELECT`
* `DISTINCT`
* `WHERE`
* `ORDER BY`
* `TOP`

## 📈 Business Insights

* Identified transactions generating losses.
* Examined sales distribution across regions.
* Analyzed purchasing behavior of corporate customers.
* Evaluated performance of technology products.

## 💻 Tools & Technologies

* Microsoft SQL Server
* SQL
* Superstore Dataset
